# 🔐 Login Page UI

A modern and responsive Login Page UI developed using HTML and CSS. This project includes a clean user interface with username and password fields, icons integration, remember me option, and stylish form design.

## ✨ Features

* User-friendly login interface
* Username and password input fields
* Font Awesome icons integration
* Remember me checkbox
* Forget password section
* Clean and modern UI design
* Responsive layout styling

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons

## 🎯 Project Objective

The purpose of this project is to practice front-end web development concepts such as form design, input styling, icon integration, and responsive user interface creation.
